# 🇫🇷 Sherlock Holmes — FRENCH Adaptation Project — Setup Guide

This is the deployable **Claude Project** for converting your finished, polished
English Sherlock scripts into **native-quality French TTS-ready scripts** —
plus French titles, description, tags, and a quiz.

It does ONE job: English script in → flawless French script out, in parts.
No SFX tags. No ledgers. No extra steps.

---

```
sherlock-french/
├── README-FR-SETUP.md                  ← this file
├── FR-CLAUDE-PROJECT-INSTRUCTIONS.md   ← paste into "Custom Instructions"
└── knowledge/                          ← upload ALL 4 to Project Knowledge
    ├── FR-01-LINGUISTIC-RULES.md       ← passé simple, vouvoiement, faux amis…
    ├── FR-02-STYLE-SAMPLE.md           ← the French VOICE anchor
    ├── FR-03-TTS-RULES.md              ← French AI-voiceover writing rules
    └── FR-04-LOCKED-GLOSSARY.md        ← names/places locked + series voices
```

---

## 🛠️ Setup (5 minutes)

1. claude.ai → Projects → **+ Create Project**
2. Name it: **"Sherlock — Adaptation Française"**
3. **Set Custom Instructions** → paste ALL of `FR-CLAUDE-PROJECT-INSTRUCTIONS.md`. Save.
4. **Project Knowledge → Add Content** → upload the 4 FR files.
5. Done.

---

## ▶️ Per-episode usage (your ONLY manual work)

1. Open a **FRESH chat** in this project.
2. Paste your **final polished English script** (the clean TTS-ready one,
   chapter-structured). Nothing else needed.
3. Claude replies with: a one-line part plan + **PART 1** (pure French).
4. Copy Part 1 between the marker lines → into your TTS.
5. Type **"next"** → get Part 2. Repeat until the last part.
6. The final reply also contains: **3–4 TITRES + DESCRIPTION + 10–12 TAGS +
   1 QUIZ** — each French item with a one-line English gloss so you always
   know what it says.

That's it. All process talk is in Hinglish/English; only the deliverables are
French.

---

## ⚠️ One-time calibration (do NOT skip — Episode 1 only)

Before publishing Episode 1, send ONE chunk (~1,500 words of the French
output) to a **native French reviewer** (Fiverr, $20–50). Ask exactly:

> "Does this read like it was originally written in French, in the register of
> a classic Sherlock Holmes translation? Flag any unnatural phrasing, modern
> slang, or grammar slips."

Feed any corrections back to Claude in the project chat and say:
*"in corrections ke basis pe FR-01 / FR-04 ke liye naye rules likh do"* —
then paste those rules into the files. After this one pass, the system is
self-sufficient. (This is exactly how your French explainer guide reached
21 rules.)

---

## 📏 Part math (for your reference)

- French runs **~15–20% longer** than English (foisonnement — normal, good).
- Each part ≈ **3,500–4,500 French words**, cut ONLY at chapter boundaries.
- 10k EN ≈ 11.5–12k FR ≈ **3 parts** · 15k EN ≈ **4 parts** · 20k EN ≈ **5–6 parts**.
- Claude announces the plan before Part 1, so you always know how many
  "next"s are coming.

---

## 🔒 Three hard guarantees built into the system

1. **Names never translate.** Holmes, Watson, Baker Street, your original
   characters — locked in FR-04. "Rue de Boulanger"-type errors are treated
   as critical failures.
2. **Story never changes.** Every clue, deduction, and plot beat is 1:1 with
   your English script. Sentence-level freedom, zero story-level freedom.
3. **Clue wording stays consistent.** The French phrasing of a clue at its
   PLANT is reused verbatim at its PAYOFF — audio listeners match by ear.
